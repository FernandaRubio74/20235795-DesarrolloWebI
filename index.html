Guía 1

    En la primera guía se aprenden las maneras de incluir estilos en un archivo HTML. Además, se exploran las diferentes técnicas de estilización, como el uso de CSS en línea, estilos internos y hojas de estilo externas. También se comprende el uso de identificadores.

  • Estilos embedidos
  • Ejercicio 2
  • Centroámerica
  • Serie favorita

Guía 2

    En esta guía se trabaja de manera exhaustiva con los estilos y se explica cómo deben implementarse en HTML. Se abordarán diferentes métodos para incluir estilos, como el uso de hojas de estilo en cascada (CSS), estilos en línea y estilos internos.

  • Película
  • Tablero
  • Biografía

Guía 3

    En la guía 3 se trabaja con 'flex' y 'z-index' para comprender su funcionamiento y cómo utilizarlos. Se explorarán las propiedades de Flexbox, que permiten crear diseños responsivos y flexibles, así como el uso de 'z-index' para manejar el apilamiento de elementos en la página.

  • Oscares
  • Menú
  • 'Viajeros'
  • Cartas

Guía 4

    En esta guía se desarrollan formularios y se explica cómo estilizarlos. Por otro lado, se trabajan las dimensiones para crear un sitio responsivo.

  • Ejercicio 1
  • Ejercicio 2
  • Ejercicio 3
  • Medias queries
  • Login
  • Sitio responsivo

Guía 5

    En esta guía se introduce el lenguaje de programación "JavaScript". Se trabaja con ejemplos, para comprender como implementar HTML, CSS Y JS.

  • Ejemplo de JS

Guía 6

    Se profundiza el uso de JavaScript y el dominio en la construcción de funciones. Asimismo, se aprende a manejar eventos usando funciones para acceder a los elementos del documento web.

  • Estructuras de control
  • Funciones JS
  • Formulario

Guía 7

    Uso del DOM y profundización en uso de JavaScript.

  • Uso del DOM
